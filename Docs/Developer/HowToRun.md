# Как открыть и запустить проект

## Открытие

1. Открой Unity Hub.
2. Нажми **Add** или **Open**.
3. Выбери папку `D:\Projects\Unity\SimulatorHorseParking`.
4. Открой проект в Unity `6000.5.0f1`.
5. Дождись первого импорта пакетов и файлов.

## Запуск технической сцены

1. В окне Project открой `Assets/_Project/Scenes/Bootstrap.unity`.
2. Нажми кнопку **Play** в верхней панели Unity.
3. Управление: `WASD` - движение; мышь - обзор; `Esc` - освободить курсор.
4. Bootstrap сейчас не является демонстрацией парковки: это чистая техническая сцена для Composition Root, камеры и контроллера.

## Что именно показывается

- Камера от первого лица без рук.
- В сцене нет видимых объектов парковки, строительных лесов, земли или временных Unity-примитивов. Визуальный контент добавляется только в соответствующем этапе roadmap и только готовыми assets.
- Composition Root создаёт и передаёт зависимости через DI.

## Если сцены нет

В верхнем меню Unity выбери `Horse Parking -> Build Bootstrap Scene`. Команда создаст или пересоздаст `Assets/_Project/Scenes/Bootstrap.unity`.
