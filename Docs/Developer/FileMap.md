# Карта файлов разработчика

Этот документ - живая карта проекта. После каждой выполненной задачи он обновляется вместе с TXT-журналом в `Docs/TaskLogs/`.

## Текущий статус

- Unity-проект создан на Unity `6000.5.0f1`.
- Каноническая карта Unity-проекта перенесена в `Assets/_Project/Docs/Developer/FileMap.md`.
- Этот файл сохраняет только навигацию по корневой документации.

## Документация

| Путь | Назначение | Где менять |
|---|---|---|
| `Docs/Roadmap_SimulatorHorseParking.docx` | Главный roadmap: этапы, архитектурные правила, правила assets и разработки. | Добавлять согласованные решения и менять статус этапов. |
| `Docs/Developer/FileMap.md` | Навигация по корневой документации. | Обновлять, если меняется расположение проектной документации. |
| `Assets/_Project/Docs/Developer/FileMap.md` | Главная developer-карта Unity-кода, assets, настроек и точек изменения. | Обновлять после каждой задачи реализации. |
| `Docs/TaskLogs/` | TXT-журналы завершённых задач: сделанные изменения, проверки, ограничения и следующий шаг. | Новый файл после каждой задачи. |

## Структура Unity

Код и assets организованы в `Assets/_Project/` по правилам Clean Architecture, зафиксированным в roadmap. Подробности - в канонической карте Unity-проекта.
